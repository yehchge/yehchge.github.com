PHP印出陣列中被包在StdClass內的Value

May 12th, 2011

 假設今天收到的資料的是
Array (
   [0] => stdClass Object (
   [name] => “AAA”
   [code] =>123
   )
[1] => stdClass Object (
   [name] => “BBB”
   [code] =>456
  )
[2] => stdClass Object (
   [name] => “CCC”
   [code] =>789
  )
)

要印出資料只要設定一個變數指到那個Array Object 就好了…
//先把資料存到某個變數 , 假設上面那筆資料是 $a
$b = $a;
//接著逐一印出的話
foreach ($b as $value) {
    $name = $value->name;
    $code = $value->code;
    echo “$name , $code \n”;
}
