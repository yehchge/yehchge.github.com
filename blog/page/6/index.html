喵喵的故事

Jan 8th, 2010

今天, 我們家的小狗狗(約克夏)因為出去散步,又下雨天,不小心被車子撞死了,家中的人都非常的心痛,我們不知道要如何處理?? 後來,請寵物醫院幫我們找了一家公司,幫我們處理喵喵之後的事情,我今天本來想要幫他挖土埋起來,但是我覺得燒一燒比較乾淨,我想對他也比較好吧!!

我不知道為什麼,就是哭不出來,我也不知道要如何哭吧! 就讓時間慢慢的去淡化這件事, 我覺得自己要堅強一點, 最近心情可能比較不好,大家就忍一下吧!

希望他在上帝的前面,可以高高興興的跑,高高興興的跳,高高興興的笑,高高興興的吃東西,我們家的寶貝!!再見了，喵喵!!

記得喵喵第一次來我們家,我們非常的高興,家中第一次養寵物, 他看起來很可愛,但是小時後常常生病,我們還要餵他吃藥藥,他也有點怕冷.我們家,我妹妹最疼他了,他還幫我們家抓到一隻老鼠,把他咬到沒辦法跑,他太厲害了,我們家10幾年來在我們家的禍害(老鼠)就這樣被他逮到了! 厲害吧

喵喵最喜歡到公園的草地上,太陽大的時候,在地上打滾, 我還曾經帶他去爬山,他很厲害喔! 爸爸也曾經帶他去台南看奶奶,聽說叫的很大聲喔!!

媽媽最傷腦筋了,因為帶喵喵回去住了一天,那一天晚上, 媽媽被喵喵吵的和我們說,他怕了,這樣和他住一天,根本沒有辦法入睡!!隔天立刻把他帶回來!!

喵喵最兇了,以前常常要幫他鏈起來,他都不給我們鏈,還會咬我們呢!妹妹就是這樣被他咬了一條!!

給永遠可愛的喵喵 94.1.13

去花蓮玩

Jan 8th, 2010

8月底,和公司的人,一起去員工旅遊,公司的其他門市的會計(4個人),和我一同做莒光號到花蓮,到達花蓮後,就有飯店(美崙大飯店)的車子載我們五個人(加上一個從彰化來的資深員工)一起過去,我們到了飯店後,就開始準備飯店的Check in,進去飯店後, 我們就開始休息,之後去看看飯店的設施,然後就詢問了一下後,本來要下午去海洋公園玩的(花蓮海洋世界,上面還有遠來飯店),後來聽說要預約, 所以就沒有辦法去了,只能明天上午去,所以開始安排飯店內的活動,按摩,逛逛商店,到游泳池玩水等,玩完後,就吃晚餐了,飯店的晚餐還不錯,可以預約,我們也吃了很多,也拍了很多照片,之後,四個由台北來的女孩子,就自行去租了摩托車,騎車去玩了,聽說去七星潭,晚上夜景很漂亮,她們晚上還跑到遠來飯店去看她們飯店的設施,很棒;(摩托車租一天要300元), 然後,她們也去花蓮逛夜市,我們3個男生,晚上就跑去吃鴨肉麵,很好吃,(第3個男生是花蓮店的會計).晚上我們4個男生,小喝一點酒,聊了很晚才去睡覺.

隔天一早,我們吃完早餐之後,就請飯店載我們過去海洋公園玩,結果一早過去,還沒有開門,只有我們3個男生先過去, 女生們會騎摩托車過去, 所以等到開始後,我們就進去逛了一下後,就跑去看海洋公園所安排的秀了,有美人魚秀,海豚秀,歌舞秀,等等.還有剛開始進去會有一些步偶歡迎我門進去,和她們拍照,我們就坐著園區內的纜車上上下下的,就像是電視劇內有人坐著纜車去滑雪的感覺差不多!!

我們進去後,看到的每一個主題樂園都是一個藝術品,都好漂亮好可愛喔, 所以就跑去拍那一些照片了.

我們去看美人魚秀的時候,發現哈薩克人的憋氣的時間很長,在水裡面表演,好棒喔,我們還有去看水上芭雷,也是很漂亮,但是看完後,好熱喔,就趕緊去找販賣機投飲料,才能舒緩一下熱度;去了那裡,我第一件事,就是幫豆豆買他說的一件有夜光功能的雙海豚的4號兒童T-Shirt, 我自己也像發了瘋似的,買了很多的紀念品,之後一點多時,本來還要去遠來飯店看看的,後來飯店打電話來說要退房,我們就趕緊回去,把房間退出,以免再被加半天的錢.

之後,我們就準備分道揚鑣,回家了!! 我們回來坐的是自強號,所以比較快一點, 這一次的員工旅遊,自行負擔了1千多塊錢,也不少喔..

(PS:相片的部分,還在照相館中,還沒有拿回來,拿回來後,就會放到網站上了)

PHP实现文件安全下载

Jan 8th, 2010

PHP实现文件安全下载
你一定会笑我”下载文件”如此简单都值得说？当然并不是想象那么简单。例如你希望客户要填完一份表格，才可以下载某一文件，你第一个想法一定是用 “Redirect”的方法，先检查表格是否已经填写完毕和完整，然后就将网址指到该文件，这样客户才能下载，但如果你想做一个关于”网上购物”的电子商务网站，考虑安全问题，你不想用户直接复制网址下载该文件，笔者建议你使用PHP直接读取该实际文件然后下载的方法去做。程序如下：

$file_name = “info_check.exe”;
$file_dir = “/public/www/download/”;
if (!file_exists($file_dir . $file_name)) { //检查文件是否存在
echo “文件找不到”;
exit;
} else {
$file = fopen($file_dir . $file_name,”r”); // 打开文件
// 输入文件标签
Header(“Content-type: application/octet-stream”);
Header(“Accept-Ranges: bytes”);
Header(“Accept-Length: “.filesize($file_dir . $file_name));
Header(“Content-Disposition: attachment; filename=” . $file_name);
// 输出文件内容
echo fread($file,filesize($file_dir . $file_name));
fclose($file);
exit;}

而如果文件路径是”http” 或者 “ftp” 网址的话，则源代码会有少许改变，程序如下：

$file_name = “info_check.exe”;
$file_dir = “http://www.easycn.net/”;
$file = @ fopen($file_dir . $file_name,”r”);
if (!$file) {
echo “文件找不到”;
} else {
Header(“Content-type: application/octet-stream”);
Header(“Content-Disposition: attachment; filename=” . $file_name);
while (!feof ($file)) {
echo fread($file,50000);
}
fclose ($file);
}

Taskbar Repair Tool Plus! (工作列修復工具)
http://www.kellys-korner-xp.com/taskbarplus!.htm
或參考微軟技術支援文章 - 登入 Windows 時找不到工作列
http://support.microsoft.com/default.aspx?scid=kb;zh-tw;318027
==================================================
把Taskbar Repair Tool Plus!下載到電腦後，點兩下
會出現一個視窗說明這是「未註冊」的版本，所以有些功能不能使用。
1.在Taskbar Problems選Minimized Programs Missing
2.點右方的「Repair」按紐
3.跳出一個視窗問您要不要執行修復工作列的程序，請選「是」
4.這時您的螢幕會閃幾下，這是正常的現象請不要擔心
5.您會發現「開始」右邊的「快速啟動」，還有右下角「時鐘」左邊的「輸入法」按紐都不見了
6.請在工具列上按「右鍵」，選第一個「工具列」→「快速啟動」，這樣您「開始」右邊的「快速啟動」就會再度出現了
7.點左下角的「開始」→「控制台」→「日期、時間、語言和區域選項」
→「地區及語言選項」→上方的標籤「語言」→「詳細資料」
→下方喜好設定的「語言工具列」→「在桌面顯示語言列」前面的方框請按一下，亦即變成「沒有」打勾的狀態→「確定」→
下方喜好設定的「語言工具列」→「在桌面顯示語言列」前面的方框請按一下，亦即變成「有」打勾的狀態→「確定」→「確定」→「確定」
8.右下角「時鐘」左邊的「輸入法」按紐又出現了
9.以後如果又發生同樣狀況也是同樣的步驟照做。建議：(工作列修復工具)使用完不要刪除，因為有可能又會發生同樣的狀況
