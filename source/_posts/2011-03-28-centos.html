CentOS 的服務版本查詢指令:

1. 查看 Linux Kernel 版本
$uname -r

2. 查看 CentOS 版本
$cat /etc/redhat-release

3. 查看 PHP 版本
$php -v

4. 查看 MySQL 版本
$mysql -v

5. 查看 Apache 版本
$rpm -qa httpd
